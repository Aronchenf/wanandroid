## Kotlin+JetPack+协程实现的MVVM架构Wanandroid客户端

- 项目基于Kotlin语言，采用ViewModel+LiveData实现MVVM架构。
- 网络请求采用协程，数据库采用Room。
- 界面黑白配色，优雅美观，适合阅读。
- 适配了白天黑夜模式。
